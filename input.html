FWP P1272 P1273 TAOcs qualification of outer depo shield cover for Yttria defect improvement

Inline Defect segmentation and EDX has indicated TAOcs as a key contributor to Yttrium type defects within the P1272/73 loop, particularly at M0FC. The lower part of the TAOcs chamber comprises of an Aluminium Outer Depo Shield which is coated in Yttrium. It has a part life of 42k wfrs in TAOcs - all VCN Operations are restricted from running on a chamber with an ODS over 25k wfrs while all other layers run within 0-42k. This WP aims to qualify the ODS Quartz Cover for P1272/73 Process â€“ this Cover is an extension of the current Inner Quartz Depo Part. No change is occurring to the current VCN/All other layer wfr Run Windows.
